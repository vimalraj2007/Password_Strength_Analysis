# 🔐 Password Strength Analysis

## 📌 About
This project focuses on evaluating password strength using an online password checker. Different passwords with varying complexity levels were created and tested to understand how password security works.

---

## 🎯 Objective
- Analyze password strength  
- Understand weak vs strong passwords  
- Learn password security best practices  

---

## 🛠️ Tools Used
- Online Password Strength Checker (PasswordMeter)  

---

## 🧪 Work Done
- Created multiple passwords (weak → strong)  
- Tested each password using a strength checker  
- Recorded scores and feedback  
- Compared results based on complexity and length  

---

## 🔍 Key Findings
- Weak passwords are easily guessable  
- Common words and patterns reduce security  
- Adding symbols and numbers improves strength  
- Longer passwords are more secure  
- Random passwords are the strongest  

---

## ⚠️ Attacks Learned
- Brute Force Attack  
- Dictionary Attack  

---

## 🛡️ Best Practices
### ✅ Do:
- Use long and complex passwords  
- Combine uppercase, lowercase, numbers, symbols  
- Use passphrases  
- Enable multi-factor authentication  

### ❌ Don’t:
- Use common words or personal info  
- Use simple patterns (name + year)  
- Reuse passwords
